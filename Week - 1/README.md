# Objectives

* Install XAMPP development environment.
* Configure Apache and MySQL services.
* Verify localhost functionality.
* Create and test a simple PHP application.
* Establish a database connection using PHP and MySQL.

## Activities Performed

### 1. XAMPP Installation

XAMPP was installed successfully to provide Apache and MySQL services required for local web development.

### 2. Apache and MySQL Configuration

Both Apache and MySQL services were started and tested through the XAMPP Control Panel.

### 3. Localhost Testing

A browser test was conducted using localhost to confirm that the Apache web server was functioning correctly.

### 4. Hello World Test

A simple PHP file was created and executed successfully to verify PHP integration with the Apache server.

### 5. Database Connection Test

A PHP database connection script was created to establish communication between PHP and MySQL. The connection was tested successfully.

## Technologies Used

* XAMPP
* Apache Server
* MySQL Database
* PHP
* Web Browser

## Outcome

The local development environment was successfully installed and configured, providing a foundation for the development of the AgriSmart Farm Management System.
