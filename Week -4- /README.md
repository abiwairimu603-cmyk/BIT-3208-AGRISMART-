# Week 4: Server-Side Programming and User Authentication

## Overview

This week focused on implementing server-side functionality for user authentication and login processing in the AgriSmart system.

## Activities Completed

### 1. Login Form Implementation

A login form was created to allow users to access the system securely.

### 2. Form Validation Testing

Client-side and server-side validation was applied to ensure correct user input during login.

### 3. PHP Login Processing

PHP scripts were developed to process user login credentials and authenticate users.

### 4. Folder Structure Setup (VS Code)

A structured project folder was organized in VS Code to separate frontend, backend, and assets for better development workflow.

### 5. Database Connection After Login

A database connection was established to verify user credentials and allow secure access after successful login.

## Outcome

A functional login authentication system was developed, enabling secure user access and reinforcing backend integration with the database.
